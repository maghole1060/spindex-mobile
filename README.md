# Spindex Mobile App - Flutter 📱

تطبيق خدمة مساعدة الطريق الذكية "Spindex" - تطبيق موبايل احترافي مبني بـ Flutter

## المميزات ✨

- 📱 واجهة مستخدم عربية رائعة وجذابة
- 🚨 زر طوارئ SOS فوري مع تأثيرات بصرية
- 🛠️ خدمات متعددة (بطارية، بنزين، كاوتش، سحب، فتح، شحن)
- 🗺️ خريطة توقع الموقع الحالي
- 👨‍🔧 تتبع الفني في الوقت الفعلي
- 💬 دردشة والمكالمة مع الفني
- 💰 خيارات دفع متعددة (كاش، فودافون كاش، كارت)
- ⭐ نظام تقييم الخدمات مع التعليقات
- 📊 إدارة حالة قوية باستخدام Provider

## البدء 🚀

### المتطلبات
- Flutter 3.0+
- Dart 3.0+
- Android Studio / Xcode
- Git

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/maghole1060/spindex-mobile.git
cd spindex-mobile

# تثبيت المكتبات
flutter pub get

# تشغيل التطبيق على محاكي أو جهاز فعلي
flutter run
```

## البنية المشروع 📁

```
lib/
├── main.dart                      # الملف الرئيسي للتطبيق
├── models/
│   └── app_state.dart            # إدارة حالة التطبيق (Provider)
├── screens/
│   └── home_screen.dart          # الشاشة الرئيسية
└── widgets/
    ├── header_widget.dart        # رأس الصفحة مع الشعار
    ├── sos_button.dart           # زر الطوارئ SOS
    ├── services_grid.dart        # شبكة خدمات المساعدة
    ├── map_widget.dart           # عرض الخريطة
    └── tracking_panel.dart       # لوحة تتبع الفني والتقييم
```

## المكتبات المستخدمة 📦

```yaml
- provider: ^6.0.0              # إدارة الحالة
- google_maps_flutter: ^2.5.0   # الخرائط
- font_awesome_flutter: ^10.5.0 # الأيقونات
- firebase_core: ^2.24.0        # Firebase
- firebase_auth: ^4.14.0        # المصادقة
- cloud_firestore: ^4.14.0      # قاعدة البيانات
- firebase_messaging: ^14.6.0   # الإشعارات
```

## الميزات الرئيسية 🎯

### 1. زر SOS الطوارئ 🚨
- يظهر مع تأثير نبضي متحرك
- يرسل إشارة فوري للطوارئ والمرور والإسعاف

### 2. اختيار الخدمة 🛠️
- 6 خدمات مختلفة مع أيقونات واضحة
- عرض السعر التقريبي لكل خدمة
- تأثيرات بصرية عند الاختيار

### 3. تتبع الفني 👨‍🔧
- عرض معلومات الفني المكلف
- وقت الوصول التقريبي
- زر المكالمة والدردشة

### 4. طرق الدفع 💰
- كاش
- فودافون كاش
- كارت ائتمان

### 5. نظام التقييم ⭐
- تقييم من 1-5 نجوم
- إضافة تعليق على الخدمة
- حفظ التقييم للخدمة

## كيفية الاستخدام 📖

1. **اختر نوع المساعدة** من الخدمات المتاحة
2. **اضغط على "اطلب مساعدة الآن"** لإرسال الطلب
3. **تتبع الفني** على الخريطة والمعلومات المعروضة
4. **تواصل مع الفني** عبر المكالمة أو الدردشة
5. **اختر طريقة الدفع** وقيّم الخدمة
